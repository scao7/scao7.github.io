Human pose researches: skeleton, SMPL and humannerf

This article is about some useful information about human pose estimation

September 25, 2022 Shengting Cao

3 minutes read

Human pose estimation is a widely studied in computer vision areas, this article reviews the detail of popular human pose estimation pipleline and methods.

OpenPose

OpenPose is a widely used framework for 2D pose estimation. The github link is: https://github.com/CMU-Perceptual-Computing-Lab/openpose. From 2016, this team published 4 different paper about how to estimate the 2D poses.

Convolutional pose machines

paper link: convolutional pose machines

This paper shows a systematic design of how convolutional network can be incorporate into the pose machine framework.

Pose Machines problem is defined as following:

p-th anatomical landmark: $$Y_p \in \mathbb{Z} \subset \mathbb{R}^2, \mathbb{Z} \ is \ the \ set \ of\ all\ (u,v)\ locations\ in\ an \ image \ Y= (Y_1,…,Y_P)$$

Pose machine: multi-class predictors

$$g_t(\cdot)\ is \ trained \ to \ predict \ the \ locations \ of \ each \ part \ in \ each \ level \ of \ hierachy $$

Feature extracted from location z denoted by $$\mathbf{x}_z \in \mathbb{R}^d$$ and contextual information from the precceding clasifier in the neighborhood. the otuput is based on the final belief values.

$$g_1(\mathbb{x}_z) \rightarrow \{b_1^p (Y_P = z) \}_p $$

$$ b_1^p (Y_P = z) $$ is the belief of each joints at stage 1.

respectively the collection of belief is shown below (P parts include the one for background).

$$ b_t \in \mathbb{R}^{w \times h \times (P+1)}$$

All the estimation of joints locations are based on 1. feature of image data and 2. contextural information from the preceeding classier in the neighborhood around each $$Y_p$$

The final equaiton is where the $$\psi_t >1(\cdot)$$ is a mapping from the beliefs $$b_{t-1}$$ to context features.

The old way to achieve the above method is using handcrafted feature with boosteed random forest. The method used in this paper replaec the hand crafted feature to convolutional neural networks.

Realtime Multi-Person 2D Pose Estimation using Part Affinity Fields

Paper link: https://arxiv.org/pdf/1611.08050.pdf

This paper introduce a idea called Part Affinity Fields (PAFs), to learn to associate the body parts with inividuals in the image. This one focused on the mutiperson scenerio. The piepline is to first take entire image as the intput of a two-branch CNN to jointly predict confidence maps for body part deteciton, and then use the affinity fields for parts association. Then the biparite matching associate the body parts candidates. Last, they assemble them into the fully body poses.

Folowed by the previous paper, a feed-forward network simultaneously predits a set of 2D confidence maps of body part locations and a set of 2D vector feilds L for part affinities. The part affinity fileds preserves both location and orientation information across the region of support of the limb.

Hand Keypoint Detection in Single Images using Multiview Bootstrapping

Paper link: https://arxiv.org/pdf/1704.07809.pdf

This paper is focusing on predict the hand pose.

OpenPose: Realtime Multi-Person 2D Pose Estimation using Part Affinity Fields

Paper link: https://arxiv.org/pdf/1812.08008.pdf

This is the final journal paper for the overall OpenPose framework.

SMPL human model

Besides the 2D skeleton estimation, a more advanced pose estimation involving human shapes is proposed by Max Planck Institute called SMPL model.

The original reserach is here: https://smpl.is.tue.mpg.de/

The idea is that the 2D skeleton can not represent the fine huamn pose. For example, it couldn’t represent the shape of human. This article will focus on the newest SMPL prediction models.
