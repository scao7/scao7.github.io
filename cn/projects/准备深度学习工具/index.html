深度学习工具tensorflow介绍以及安装

Windows 上 anaconda，tensorflow的安装，以及如何调用英伟达GPU

June 27, 2021 Shengting Cao

1 minute read

现在深度学习已经越来越火，身边的不少人都有深度学习编程都需求，为了方便大家学习同时也尽量避免我一个一个的教他们安装。这个博客会一步一步的教大家怎么在Windows上安装上Anaconda， tensorflow-gpu。

Anaconda(https://www.anaconda.com/) 是一个python包的管理软件，他能帮助你一键创建你的python运行环境并管理python相关的第三方模块。

进入官网安装anaconda Windows版

在anaconda 安装好之后我们在Windows 开始菜单搜索anaconda,我们会发现有两个终端程序可以打开。 第一种是Anaconda Powershell Prompt, 另一个是Anaconda Prompt. Anaconda Powershell Prompt 的指令是类似Linux终端的指令，而Anaconda Prompt是类似Dos系统的操作指令。我一般使用Anaconda Powershell Prompt.

打开Anaconda Powershell Prompt,开始以下指令

(cheatsheet下载)

conda create -n 你的环境名 python=你想安装的python版本 #创建环境
conda activate 你的环境名 #进入你的环境

因为现在深度学习相关内容版本更行很快所以导致有很多项目之间版本不兼容，最好的办法就是用以上的指令为不同的项目创建不同的环境

检查电脑是否有GPU以及是否正确安装好了驱动

nvidia-smi # 检查GPU

如果你的电脑有GPU并且驱动正常，你会得到下面类似的信息,如果没有首先检查电脑有没有安装GPU并且更行GPU驱动。注意到你的右上角有个CUDA version 11.3. 如果我们想要tensorflow支持GPU我们需要下载并且安装两个工具：CUDA toolkit 和cuDNN.

安装CUDA toolkit

官网下载：https://developer.nvidia.com/cuda-toolkit 选择和你GPU适配的版本，我的是11.3 所以需要安装11.x 版本。 安装选项全选默认即可。

安装cuDNN库

官网下载 https://developer.nvidia.com/cudnn 下载cuDNN需要注册一个英伟达账户。 下载下来后是一个zip压缩文件包，解压文件包并且复制里面的所有文件到CUDA toolkit 目录下。

目录位置在

C:\Program Files\ NVIDIA GPU Computing Toolkit\CUDA\v11.3\

安装tensorflow-gpu 并检查gpu是否用上

conda activate 你的环境名 #进入你的环境
pip install tensorflow-gpu # 使用pip安装tensorflow gpu版本
python # 进入python对话式编程
import tensorflow #导入tensorflow包
