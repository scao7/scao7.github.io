如何制作HugoPage

这一篇文章是讲解如何用Hugo网页自动生成器来做一个这样的博客网址。并且部署到Github上。

下载Hugo以及安装

工具：Hugo 下载界面 ,Homebrew

从上面的链接点进去就到来Hugo的下载,我使用的是Mac系统的安装方法。Mac 是通过brew来获取安装工具的所以点击上面Homebrew 链接来进行安装如果你之前没有安装过brew的话。


brew install hugo

hugo version

hugo new site yourProject # 你的工程文件名

cd yourProject

git init

git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke # 自行在github 上找主题，没有主题文件hugo是不能创建网页的。或者你可以自己创作主题。

echo theme = "ananke" >> config.toml #将主题名写入config文件。

在Hugo的下载界面有详细的教程来指导你如何安装并使用Hugo。我这里就不重复的说明了。

编辑网页

打开文件的structure，每一个主题都会有自己的格式的控制每一个post显示的内容。Hugo的网页内容基本是用.md 后缀的文档来渲染网页的。 首先用hugo new posts/my-first-post.md 这个指令来创建一个新的文章。 用你喜欢的编辑器打开，然后加入文章标题等一些header的信息。我使用的主题的header信息就是以下图片所显示的。

下面这张截图来告诉我们如何去寻找这种header的信息 在themes —- yourtheme —- exampleSite —- content —- blog 下找到一个示范的.md文件。通过查看他就可以找到如何为你的文章添加标签。或者你可以直接把content文件夹复制粘贴到yourProject 文件夹下。然后通过修改.md来生成自己想要的内容。

如何编辑网页内容就需要参考markdown 的各种文档了。

编辑网页跟渲染

打开terminal在yourProject文件夹下输入指令hugo server 然后你就可以在网页上输入链接 http://localhost:1313 来进入网页的preview。

使用Github来host你的网站

工具：Github Page

通过以上Github Page 的教程页来对Github Page有个基本的了解。然后我们指导只要把html文档放到Github上然后设置Github pages. 那么我们的Hugo是一个网页生成器通过hugo指令我们的文件会多一个文件夹public。Public 下面存放的就是我们网址所以的html文件。只要你把这个里面的文件放到github上就可以直接用github page 的链接来打开这个网站了。

当然你肯定也想保存一下你的工程文件到GitHub。所以最简单的做法就是需要两个Github仓库来分别存放我们的Hugo工程文件和生成的html文件。当你成功的config好这两个文件之后，我写来一个简单的shell文档来实现一键部署你的网站。

将一下代码复制，创建一个deploy.sh 的文档然后将以下代码放到Hugo工程文件夹下。你可能需要chmod +x deploy.sh来给他权限。 当每次编辑完你的网站之后你可以使用./deploy.sh来一键上传到github了。

hugo -d ../scao7.github.io/ # scao7.github.io 是我的Github Page 通过这个网站也可以访问我的博客。
git add .
git commit -m "update"
git push
cd ../scao7.github.io/
git add .
git commit -m "update by automation"
git push
cd ../MyNoteBook/ #MyNoteBook 是我的文件夹用来存放所以Hugo工程文件的文件夹。
