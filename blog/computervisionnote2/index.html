LeNet-5

Here is a example of neural network:

First we have a 32 * 32 * 3 input image, we perform a convolute using f = 5 and s = 1 then we do a maxpooling using f = 2 s = 2. This is called first layer. then we apply the same thing as the second layer. after go through the second layer we convert the matrix to a vector with 400 numbers and connect it to a fully convolutional layer and then do a second fully convolutional layer. Last step we do a softmax to get the clarification result.

This is a kind of CNN.

This image shows the network learn the most parameters in the fully convolutional part.Activation size go smaller and smaller.

advantages of convolutions

Parameter sharing: A feature detector that’s useful in one part of the image is probably useful in another part of the image.

Sparsity of connections: in each layer each output value depends only on a small number of inputs
